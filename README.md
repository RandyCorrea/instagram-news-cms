# Instagram News CMS - App de Noticias Tipo Instagram

Una aplicacion web moderna tipo Instagram con CMS integrado y hospedaje automatico en GitHub Pages.

**Repositorio**: https://github.com/RandyCorrea/instagram-news-cms

## Inicio Rapido

```bash
git clone https://github.com/RandyCorrea/instagram-news-cms.git
cd instagram-news-cms
npm install
npm run dev
```

Ves a: http://localhost:3000
CMS Admin: http://localhost:3000/admin-secret-access

## Features

✓ Feed tipo Instagram (mobile-first)
✓ CMS integrado con autenticacion
✓ Bloq por intentos fallidos (30 dias)
✓ SEO optimizado (Open Graph + Twitter Cards)
✓ Compartir en redes sociales
✓ GitHub Pages hosting automatico
✓ GitHub Actions CI/CD

## Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Markdown
- GitHub Pages
